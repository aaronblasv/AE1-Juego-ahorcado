# Juego del Ahorcado

Juego clásico del ahorcado desarrollado en PHP como proyecto educativo para la asignatura de Despliegue de Aplicaciones Web.

## Instrucciones básicas

1. Al iniciar el juego, se muestra una palabra oculta representada con guiones bajos (_)
2. Introduce una letra en el campo de texto
3. Presiona "Probar letra" o Enter
4. Si aciertas, la letra se revelará en su posición
5. Si fallas, perderás un intento
6. **Ganas:** Completas la palabra antes de quedarte sin intentos
7. **Pierdes:** Se agotan los intentos sin completar la palabra

## Consejos

- Comienza con vocales (A, E, I, O, U)
- Luego prueba consonantes comunes (R, S, T, N, L)
